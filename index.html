Inheritance cascades downwards

Properties that can be inherited cascade downwards, and child elements will get a computed value which represents its parentâ€™s value. This means that if a parent has font-weight set to 300: all child elements will be light unless they have font-weight set to a different value or the user agent stylesheet has a value for font-weight, for that element.

That is the case here. All text is font-weight: 300, but <span> elements have a font-weight of 700 which overrides the inheritance. This sentence is wrapped in a <span>, so the text is bold weight.

Notice also how the inline code also inherits the bold font-weight.
